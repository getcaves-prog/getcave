Cómo Ejecutar Localmente

# Clonar repositorio git clone https://github.com/cavesapp/cavesapp.git cd cavesapp # Instalar dependencias pnpm install # Configurar variables de entorno cp .env.example .env.local # Completar con las credenciales de Supabase # Ejecutar en desarrollo pnpm dev # Abrir en navegador http://localhost:3000
